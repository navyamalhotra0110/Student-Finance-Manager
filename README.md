# Student Personal Finance Manager

A web-based application designed to help students track their income, expenses, and savings goals. It features a responsive dashboard, dynamic charts, and local storage persistence.

## Features
- **Dashboard Overview:** Real-time calculation of Total Income, Expenses, and Balance.
- **Transaction Tracking:** Add and delete income/expense items with descriptions.
- **Visual Analytics:** Interactive Doughnut chart visualizing Income vs. Expense.
- **Savings Goal Tracker:** Set a financial target (e.g., buying a laptop) and track progress visually.
- **Data Persistence:** Uses Browser LocalStorage so data is never lost on refresh.
- **Investment Education:** A dedicated section to learn about Mutual Funds, SIPs, and Stocks.

## Tech Stack
- **HTML5:** Structure and Layout
- **CSS3:** Flexbox, Grid, and Custom Properties (Variables)
- **JavaScript (ES6):** Logic, DOM Manipulation, and LocalStorage API
- **Chart.js:** External library for rendering data charts
